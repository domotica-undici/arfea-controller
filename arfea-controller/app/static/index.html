Dispositivi (porte seriali)

Assegna le porte seriali dell'host (Modbus, Z-Wave, Zigbee, Thread) ai servizi. Non serve più configurarle nel file: si aggiungono qui.

Sorgente = porta host (i nomi by-id sono stabili al riavvio). Target = come la vede il servizio (es. /dev/ttyUSB0 per Modbus, /dev/zwave, /dev/zigbee).
